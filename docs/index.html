Simulating Advanced Drone Networks

A robust platform for modeling drone communication, latency, packet loss, and more.

Key Features

Dynamic Network Simulation

Simulate real-time network conditions such as latency, jitter, and packet loss.

ZeroMQ Integration

Efficient communication through ZeroMQ for reliable message exchange.

Customizable Environments

Adjust simulation parameters for different drone scenarios.

How to Install AeroLinkSim

  1. Clone the repository: git clone https://github.com/Nakok/AeroLinkSim.git
  2. Navigate to the project directory: cd AeroLinkSim
  3. Install dependencies: pip install -r requirements.txt
  4. Run the environment: python run.py

Usage

Use the following code to start simulating drone networks:

      
from agent import PPOAgent
from env import AeroLinkSimEnv

env = AeroLinkSimEnv()
agent = PPOAgent(env)

for episode in range(100):
    agent.train()
    print(f"Episode {episode + 1} completed.")
